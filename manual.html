⛳ VR Mini-Golf

Mobile VR Game Manual — CMPT 461 Immersive Computing

1. Overview

VR Mini-Golf is a mobile VR mini-golf game built with A-Frame. Play through two unique courses using only your head movement — aim with gaze, charge power by tilting, and swing with a dwell-based reticle. No controllers needed!

Key Features:
  • Two distinct golf courses with unique themes and obstacles
  • Gaze-based aiming and head-tilt power control
  • Stereoscopic VR mode for Google Cardboard / mobile VR headsets
  • Always-visible scoreboard HUD
  • Custom textured objects (ball, turf, obstacles, sky)
  • Sound effects and background music
  • Collision detection with walls and obstacles
  • Setup menu with difficulty, audio, and control options

2. Setup & Installation

Option A: Run from a web server (recommended)

  1. Download or clone the project folder
  2. Open a terminal in the project root directory
  3. Start a local web server: 
    npx serve .
# or
python3 -m http.server 8080
  4. Open the URL in your browser (e.g., http://localhost:3000 or http://localhost:8080)

Option B: GitHub Pages

  1. Push the project to a GitHub repository
  2. Enable GitHub Pages in the repo settings (source: main branch, root folder)
  3. Access the game at https://<username>.github.io/<repo-name>/
💡 Tip: For mobile VR, open the game URL on your phone's browser (Chrome recommended), then tap the VR goggles icon to enter stereoscopic mode.

Requirements

4. How to Play

Game Flow

  1. FLASH Game starts with a flashscreen showing game title and credits
  2. SETUP Configure audio, difficulty, and controls on the setup screen
  3. PLAY Play through Hole 1 and Hole 2, trying to minimize strokes
  4. SCORE See your final score and rating at the end

Controls

Action Control Description
Aim Look left / right Turn your head (or mouse on desktop) to aim the shot direction. A yellow arrow shows your aim line.
Power Tilt head forward (Gaze+Tilt mode) In Tilt mode, tilt your device/head forward to charge the power gauge. The more you tilt, the more power.
Power Auto-charge (Gaze Only mode) In Gaze Only mode, power automatically charges while you aim. Time your swing!
Swing Gaze at SWING button Look at the green SWING circle above the ball. Hold your gaze for 1.5 seconds to execute the shot.
⚠️ No hand controls: The entire game is played with head movement only. On desktop, use the mouse to look around and click the SWING button.

Scoreboard HUD

The scoreboard is always visible at the top of your view, showing:

  • Hole — Current hole number (1 or 2)
  • Strokes — Number of swings taken on current hole
  • Total — Cumulative strokes across all holes
  • Par — Target number of strokes for this hole

5. Levels

Hole 1 — Park Course

  • Theme: Lush green park with grass turf
  • Par: 3
  • Obstacles: 2 stationary brick blocks
  • Wind: Gentle left-to-right breeze
  • Difficulty: Straight shot with minor obstacles

Hole 2 — Canyon Course

  • Theme: Sandy desert canyon
  • Par: 4
  • Obstacles: 3 blocks + 1 moving obstacle
  • Wind: Stronger, shifting wind
  • Difficulty: Longer distance, offset hole, moving barriers

6. Setup Options

Option Values Description
Background Music On / Off Toggle ambient background melody
Sound Effects On / Off Toggle swing, collision, and hole-in sounds
Difficulty Easy / Hard Hard mode increases wind strength and obstacle speed
Control Mode Gaze Only / Gaze + Tilt Gaze Only auto-charges power; Tilt mode uses head tilt for power

7. Scoring

Your goal is to complete each hole in as few strokes as possible. Each swing counts as one stroke.

Rating Condition
🌟 Amazing! 2+ strokes under total par
🏅 Great job! At or under total par
👍 Nice game! 1-2 strokes over par
⛳ Keep practicing! 3+ strokes over par

8. VR / Stereoscopic Mode

  1. Open the game on your mobile phone's browser
  2. Tap the VR goggles icon (bottom-right of the screen)
  3. Insert your phone into a Google Cardboard or compatible VR headset
  4. The screen will split into stereoscopic view for each eye
  5. Aim and play using head movements only

9. Troubleshooting

Issue Solution
No VR button visible Ensure you're using HTTPS (GitHub Pages) and a WebXR-compatible browser
No sound Tap the screen once to unlock audio (browser autoplay policy). Check Setup menu for muted audio.
Textures not loading Ensure you're running via a web server, not opening the HTML file directly (file:// protocol may block texture loading)
Laggy on mobile Close other browser tabs and apps. Reduce screen brightness to help performance.
Head tracking not working Ensure your device has a gyroscope and you've granted motion sensor permissions
